InstiCab

On a university campus, multiple Cabs, Autos, and Rickshaws are available for commuting. The students and professors use each of these to commute inside and outside the campus. However, the availability of these vehicles at a particular time and location is not guaranteed.

  • This web application facilitates both parties such that the users (i.e., people who need to use the service) would be able to find the nearby available rides, and the ride providers (i.e., the cab, auto, and rickshaw drivers) would find the available passengers very efficiently.
  • Application contains three types of interfaces - one for the users, one for the ride providers and one for the admins.
